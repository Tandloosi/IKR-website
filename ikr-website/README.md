# IKR Website

Next.js marketing site for IKnowRight.

## Setup

```bash
cd ikr-website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint

## Structure

- `src/app/` — pages (home, aanpak)
- `src/components/ikr/` — UI sections
- `src/data/videos.ts` — carousel & feed video paths
- `public/images/` — live assets
- `public/videos/` — TikTok showcase clips

Design references live in `../design/` (not deployed).
